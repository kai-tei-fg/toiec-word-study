# English Word Test Application
This application is designed for the parents who help their kids to memorize new English words.

## Features

### For Kids
- Test with word lists which is prepared by parents
- Test result can be downloaded for further test
- Prepared word lists and temporary uploaded word lists(CSV files) can be used for the test
- You can test all the words in the list or part of words in the list by using the options

### For parents
- Manage the word list
